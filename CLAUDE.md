# CLAUDE.md

Указания для Claude Code при работе в этом репозитории.

## Язык и оформление

- **Описания и заголовки Pull Request — ВСЕГДА на русском.** Это твёрдое правило
  для всех репозиториев владельца (kir-kopylov). Без исключений.
- Сообщения коммитов — на русском (в стиле существующей истории репозитория).
- Комментарии в коде и docstring'и — на русском, как и в текущем коде.
- Общение с пользователем — на русском.

## Тесты

- Прогон юнит-тестов: `pytest -m unit ./tests/` (в CI и pre-commit).
- Маркеры: `unit` (быстрые, без сети), `integration` (реальный Google Sheets API),
  `expensive` (вызовы Claude API — стоят денег).
- Перед push убедись, что `ruff format --check` и `ruff check` чистые, а
  coverage-gate (80%) проходит.
